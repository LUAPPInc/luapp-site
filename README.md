> Lunnar Applications

## Install

This site use [Gatsby](https://www.gatsbyjs.org/)

 1. Install Yarn

```js
npm install -g yarn
```

 2. Install dependences

```js
yarn install
```

 3. Start server

```js
yarn start
```
The project will run at `http: // localhost: 8000`

  4. build
```js
yarn build
```
